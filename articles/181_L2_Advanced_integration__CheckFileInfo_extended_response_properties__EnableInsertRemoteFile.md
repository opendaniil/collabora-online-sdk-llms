New in version 24.04.10.

If set to true, this will enable the insertion of files (e.g., multimedia) chosen from the WOPI storage. A [UI_InsertFile](257_L0_PostMessage_API.md) postMessage will be sent to the WOPI host to request the UI to select the file.
