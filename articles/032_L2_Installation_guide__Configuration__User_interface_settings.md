With Collabora Online, the system administrator can set the default user interface mode between a `tabbed` view, where controls are grouped into tabs that contain buttons with icons and labels, and a `compact` view, which features a minimal menu and a toolbar. Please refer to the `user_interface.mode` setting in the configuration file.

New in version 6.4: `user_interface.mode` was firstly introduced in version 6.4, albeit with different values (`classic` and `notebookbar`)

Deprecated since version 21.11: `classic` and `notebookbar` are no longer supported. Use `compact` or `tabbed` instead.
