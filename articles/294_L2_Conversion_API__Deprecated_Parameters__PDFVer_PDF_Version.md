When the output format is PDF, this parameter sets the PDF version for the respective type of PDF to be used for the output file. Example: `PDFVer=PDF/A-2b`. (Obsoleted by `SelectPdfVersion`)

| Supported values |
| --- |
| PDF-1.5 |
| PDF-1.6 |
| PDF-1.7 |
| PDF-2.0 |
| PDF/A-1b |
| PDF/A-2b |
| PDF/A-3b |
| PDF/A-4 |
