The format of the data is specified be MIME type. Valid MIME types include `text/plain` for plain text, `text/markdown` for Markdown formatted text, and `text/html` for HTML markup. The MIME type string also should include the encoding like utf-8 by appending `;charset=utf-8`.

Examples:

- Text `text/plain;charset=utf-8`
- HTML `text/html;charset=utf-8`
- Markdown `text/markdown;charset=utf-8`
