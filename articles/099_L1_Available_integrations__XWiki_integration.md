XWiki SAS provides an integration of Collabora Online with XWiki 15.10 or above to view, edit and collaborate real-time on office attachments.

You can install it directly from the [XWiki store](https://store.xwiki.com/xwiki/bin/view/Extension/Collabora%20Connector%20Application%20%28Pro%29/) [https://store.xwiki.com/xwiki/bin/view/Extension/Collabora%20Connector%20Application%20%28Pro%29/].

See the [installation guide](https://store.xwiki.com/xwiki/bin/view/Extension/Collabora%20Connector%20Application%20%28Pro%29/#installation) [https://store.xwiki.com/xwiki/bin/view/Extension/Collabora%20Connector%20Application%20%28Pro%29/#installation] for how to install and set it up.

The source code is available on [GitHub xwikisas/application-collabora](https://github.com/xwikisas/application-collabora) [https://github.com/xwikisas/application-collabora]
