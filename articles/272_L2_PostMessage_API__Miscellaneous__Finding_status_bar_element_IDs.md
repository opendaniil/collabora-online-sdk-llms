Status bar button IDs are defined in the `onDocLayerInit` function in [Control.StatusBar.js](https://github.com/CollaboraOnline/online/blob/main/browser/src/control/Control.StatusBar.js) [https://github.com/CollaboraOnline/online/blob/main/browser/src/control/Control.StatusBar.js]. Note that they usually don’t change but there is no guarantee that they are stable.
