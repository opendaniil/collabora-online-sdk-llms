The Collabora branding sheet only styles the chrome. The in-document UI — toolbars, sidebars, ruler, headers, annotations, dialogs, status badges and so on — is driven by the core stylesheets under `browser/css/` and is also overridable through `css_variables`. The same files apply to the CODE build of COOL, where the `--co-*` family above is not loaded; in that case use this list directly.

Refer to the colour palettes CSS files for the canonical defaults:

- [regular](https://github.com/CollaboraOnline/online/blob/main/browser/css/color-palette.css) [https://github.com/CollaboraOnline/online/blob/main/browser/css/color-palette.css]
- [dark](https://github.com/CollaboraOnline/online/blob/main/browser/css/color-palette-dark.css) [https://github.com/CollaboraOnline/online/blob/main/browser/css/color-palette-dark.css]
