These are the options you can set:

| Setting | Description |
| --- | --- |
| Enable plugin | Whether the plugin is enabled or not. |
| Collabora Online Address | The URL of the Collabora Online server. |
| Disable certificate verification (insecure) | Disable the TLS certificate verification when connecting to the Collabora Online server. Caution Should only be “checked” for development, if you don’t have valid certificates. |
| Token Encryption Key | This secret is used to generate the token to access the documents. You can regenerate one by clicking “Regenerate”. This will invalidate all current tokens. |
| File Edit Permissions | If true the owner can edit the file and share access. |
