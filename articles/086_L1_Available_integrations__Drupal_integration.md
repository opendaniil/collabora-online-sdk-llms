The Drupal integration allows you to edit or view in the browser office documents stored as media in Drupal. The extension adds an action Edit with Collabora Online on documents which can be opened with Collabora Online and if the user has the write permission. You can also view documents attached to a content page.

You can install the module directly like a regular [Drupal module](https://www.drupal.org/project/collabora_online/) [https://www.drupal.org/project/collabora_online/] from your Drupal project.

`collabora-drupal.git` [https://github.com/CollaboraOnline/collabora-drupal]

```
  composer require 'drupal/collabora_online:^0.9@beta'
```

[More info on Drupal integration](https://www.collaboraonline.com/integrations/drupal/) [https://www.collaboraonline.com/integrations/drupal/]
