This guide covers two deployment approaches:

1. Standard COOL/CODE Deployment: COOL/CODE deployment using standard Helm chart.
2. COOL with COOL Controller Deployment: COOL deployment with [COOL Controller](https://www.collaboraonline.com/collabora-online-controller/) [https://www.collaboraonline.com/collabora-online-controller/] using umbrella Helm chart.
