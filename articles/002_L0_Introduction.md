# What is Collabora Online?

Collabora Online is a web based office documents editor, based on LibreOffice, and compatible with the major office suites on the market. You can view and edit text document, spreadsheet, presentations and diagrams, as well as PDF, and collaborate with others.

Collabora Online isn’t a standalone solution, it requires an EFSS (Enterprise File Synchronization and Sharing) or any other system to manage the documents and user permissions.
