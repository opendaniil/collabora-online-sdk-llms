The Collabora Online module for the ERP and CRM software Odoo 17, 18 and 19 allow viewing and editing office documents attached to Odoo content.

`Download from the Odoo app store` [https://apps.odoo.com/apps/modules/19.0/collabora_odoo]

[How to set it up](115_L1_Integrations_setup__Odoo.md)

The source code is available on [GitHub CollaboraOnline/collabora_odoo](https://github.com/CollaboraOnline/collabora_odoo/) [https://github.com/CollaboraOnline/collabora_odoo/]
