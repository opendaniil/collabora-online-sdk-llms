| Document Type | Output Formats |
| --- | --- |
| Writer | doc, docm, docx, fodt, html, odt, ott, pdf, rtf, txt, xhtml, png |
| Calc | csv, fods, html, ods, ots, pdf, xhtml, xls, xlsm, xlsx, png |
| Impress | fodp, html, odg, odp, otp, pdf, potm, pot, pptm, pptx, pps, ppt, svg, swf, xhtml, png |
| Draw | fodg, html, odg, pdf, svg, swf, xhtml, png |

Note

Of course not every input/output combination makes sense or works at all. Be reasonable. See the examples section below.
