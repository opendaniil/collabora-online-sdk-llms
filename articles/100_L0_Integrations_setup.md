# Integration of Collabora Online

Collabora Online needs another software component to work, providing the file storage and user management. This component is usually an [EFSS](https://en.wikipedia.org/wiki/Enterprise_file_synchronization_and_sharing) [https://en.wikipedia.org/wiki/Enterprise_file_synchronization_and_sharing] but it can also be a CMS or any other. The glue that connect things together is referred to as the integration and implements WOPI as described in [How to integrate](146_L0_How_to_integrate.md).

This chapter will explain how to use existing integrations from a deployment point of view and is aimed at system administrators.
