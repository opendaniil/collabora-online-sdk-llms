The [Collabora Online for WP](https://wordpress.org/plugins/collabora-online/) [https://wordpress.org/plugins/collabora-online/] is a WordPress™ plugin that integrates Collabora Online to allow you to collaborate on office documents with other users.

You can install the plugins directly like a regular [WordPress™ plugin](https://wordpress.org/plugins/collabora-online/) [https://wordpress.org/plugins/collabora-online/] from your WordPress™ installation.

[How to set it up](124_L1_Integrations_setup__WordPress.md)

The source code is available on [GitHub CollaboraOnline/collabora-wordpress](https://github.com/CollaboraOnline/collabora-wordpress/) [https://github.com/CollaboraOnline/collabora-wordpress/]
